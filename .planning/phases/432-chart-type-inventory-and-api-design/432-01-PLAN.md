   - Data model: RibbonChartData with IReadOnlyList<RibbonSeries> Series, ScatterChartMetadata Metadata
   - RibbonSeries: IReadOnlyList<ScatterPoint> Points, float Radius, uint Color, string? Label

   - VectorFieldRenderArrow: readonly record struct with Vector3 Position, Vector3 Direction, double Magnitude, uint Color
   - VectorFieldProbeStrategy: nearest-point probing returning magnitude and direction
   - VectorFieldChartRenderingStatus: sealed record with HasSource, IsReady, ArrowCount, MagnitudeRange, ViewSize

4. **Heatmap Slice Chart API Contract** -- Complete type signatures:
   - Plot3DAddApi: `HeatmapSlice(double[,] values, HeatmapSliceAxis axis, double position, string? name = null)` and `HeatmapSlice(HeatmapSliceData data, string? name = null)`

   - HeatmapSliceAxis: enum { X, Y, Z }
   - HeatmapSlicePlot3DSeries: SetPosition(double position), SetColorMap(SurfaceColorMap colorMap)
   - HeatmapSliceRenderer.BuildScene(HeatmapSliceData data) returning HeatmapSliceRenderScene
   - HeatmapSliceRenderScene: IReadOnlyList<HeatmapSliceRenderCell> Cells, HeatmapSliceAxis Axis, double Position
   - HeatmapSliceRenderCell: readonly record struct with Vector3 Position, Vector2 Size, uint Color
   - HeatmapSliceProbeStrategy: bilinear interpolation probing
   - HeatmapSliceChartRenderingStatus: sealed record with HasSource, IsReady, CellCount, Axis, Position, ViewSize

5. **Box Plot Chart API Contract** -- Complete type signatures:
   - Plot3DAddApi: `BoxPlot(BoxPlotData data, string? name = null)`
   - Data model: BoxPlotData with IReadOnlyList<BoxPlotCategory> Categories
   - BoxPlotCategory: string Label, double Min, double Q1, double Median, double Q3, double Max, IReadOnlyList<double> Outliers, uint Color
   - BoxPlotPlot3DSeries: SetCategoryColor(int index, uint color)
   - BoxPlotRenderer.BuildScene(BoxPlotData data) returning BoxPlotRenderScene
   - BoxPlotRenderScene: IReadOnlyList<BoxPlotRenderBox> Boxes, IReadOnlyList<BoxPlotRenderWhisker> Whiskers, IReadOnlyList<BoxPlotRenderOutlier> Outliers
   - BoxPlotRenderBox: readonly record struct with Vector3 Position, Vector3 Size, uint Color
   - BoxPlotRenderWhisker: readonly record struct with Vector3 Start, Vector3 End, uint Color
   - BoxPlotRenderOutlier: readonly record struct with Vector3 Position, uint Color
   - BoxPlotProbeStrategy: category-based probing returning statistical summary
   - BoxPlotChartRenderingStatus: sealed record with HasSource, IsReady, CategoryCount, BoxCount, ViewSize

6. **Common Patterns** -- Document shared patterns:
   - All data models: sealed class, ReadOnlyCollection<T>, constructor validation

   - All probe strategies: sealed class implementing ISeriesProbeStrategy
   - All series subclasses: sealed class extending Plot3DSeries, internal constructor
