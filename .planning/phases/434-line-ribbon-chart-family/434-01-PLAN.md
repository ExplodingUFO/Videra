**LineRenderScene.cs** (per LINE-01):
- `readonly record struct LineRenderSegment(Vector3 Start, Vector3 End, float Width, uint Color)` in namespace `Videra.SurfaceCharts.Core`
- `sealed class LineRenderScene` with constructor `LineRenderScene(int seriesCount, IReadOnlyList<LineRenderSegment> segments)`
- Validation: `ArgumentNullException.ThrowIfNull(segments)`, `ArgumentOutOfRangeException.ThrowIfNegativeOrZero(seriesCount)`
- Backing: `ReadOnlyCollection<LineRenderSegment>` via `Array.AsReadOnly(segments.ToArray())`

**LineRenderer.cs** (per LINE-01):
- `public static class LineRenderer` with `static LineRenderScene BuildScene(LineChartData data)`
- `ArgumentNullException.ThrowIfNull(data)` at entry
- For each series in `data.Series`, for each adjacent pair `(i, i+1)`:
  - Create `LineRenderSegment` with `new Vector3((float)start.Horizontal, (float)start.Value, (float)start.Depth)`, same for end, `series.Width`, `series.Color`
- Return `new LineRenderScene(data.SeriesCount, segments)`

**RibbonRenderScene.cs** (per LINE-02):
- `readonly record struct RibbonRenderSegment(Vector3 Start, Vector3 End, float Radius, int Sides, uint Color)` -- Sides defaults to 8 for tube cross-section
- `sealed class RibbonRenderScene` with same pattern as LineRenderScene
- Properties: `int SeriesCount`, `IReadOnlyList<RibbonRenderSegment> Segments`

**RibbonRenderer.cs** (per LINE-02):
- `public static class RibbonRenderer` with `static RibbonRenderScene BuildScene(RibbonChartData data)`
- Same iteration as LineRenderer but creates `RibbonRenderSegment` with `series.Radius`, `Sides: 8`, `series.Color`

**LineProbeStrategy.cs** (per LINE-04):
- `sealed class LineProbeStrategy : ISeriesProbeStrategy`
- Constructor: `LineProbeStrategy(LineRenderScene scene, double snapRadius = 0.05)`
- `TryResolve`: iterate all segments, compute point-to-segment XZ distance (reuse `ContourProbeStrategy.PointToSegmentDistance` pattern), find nearest segment within `_snapRadius`, project cursor onto segment via `ProjectOntoSegment`, interpolate Y value at projection point `t`, return `SurfaceProbeInfo(sampleX, sampleY, axisX, axisY, value: interpolatedY, isApproximate: false)`. Return `null` if no segment within snap radius.
- Include private static `PointToSegmentDistance` and `ProjectOntoSegment` helper methods (copy from ContourProbeStrategy)

**RibbonProbeStrategy.cs** (per LINE-04):
- `sealed class RibbonProbeStrategy : ISeriesProbeStrategy`
- Constructor: `RibbonProbeStrategy(RibbonRenderScene scene, double snapRadius = 0.1)`
- `TryResolve`: same as LineProbeStrategy but distance threshold includes segment `Radius` (effective snap = `_snapRadius + segment.Radius`)
