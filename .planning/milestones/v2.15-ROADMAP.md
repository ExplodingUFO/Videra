# Roadmap

## Active Milestone

`v2.15 Opt-In Direct Lighting Baseline`

Goal: start the deferred advanced-runtime line with one bounded opt-in direct-lighting slice on the existing static-scene viewer/runtime path, without widening into shadows, environment maps, post-processing, animation, or a new public package line.

## Scope Guard

- No shadows, environment maps, post-processing, animation, skeletons, morph targets, or mixer/timeline work.
- No second public UI adapter/package line, no backend/platform expansion, and no broader importer/chart-family widening.
- No generic render-graph/plugin framework, no compatibility shims, no migration adapters, and no transitional dual-renderer public contract.
- No public optional-pack/package publishing changes in this milestone; keep the slice opt-in and bounded on the current repository/runtime line.
- `.planning` remains local-only and does not define a repository release boundary by itself.
