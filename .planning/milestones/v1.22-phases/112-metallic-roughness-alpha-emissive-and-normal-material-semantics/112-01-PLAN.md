# 112-01 Plan

## Goal

Add explicit static PBR material semantics to the runtime material contract and map them from static glTF import without widening into renderer/backend feature work.

## Tasks

1. Extend runtime material contracts narrowly:
   - add explicit metallic/roughness semantics
   - add explicit alpha mode / cutoff / double-sided semantics
   - add explicit emissive and normal-map-ready semantics
2. Update the glTF importer to populate those semantics from SharpGLTF material properties and channels:
   - metallic/roughness factors and texture
   - alpha mode / alpha cutoff / double-sided
   - emissive factor and texture
   - normal texture and scale
3. Add focused tests proving:
   - imported material catalogs retain the new semantics
   - the retained scene/runtime path still preserves those catalogs through `SceneImportService`
