# 111-01 Plan

## Goal

Make UV coordinates and texture-usage/color-space intent first-class runtime truth for static glTF imports, then prove the retained viewer/runtime path preserves that truth.

## Tasks

1. Add narrow runtime contracts for UV sets and texture usage:
   - explicit mesh UV retention on `MeshData`
   - one direct material texture-binding contract that carries texture id, sampler id, texcoord set, and color-space intent
2. Update glTF importer/texture reader to populate those contracts from current static glTF input:
   - read `TEXCOORD_*` accessors
   - map base-color texture usage onto the new binding
   - remove the hard-coded texture-level `isSrgb` ambiguity
3. Add focused tests that prove:
   - glTF import retains UV coordinates and binding metadata
   - the retained imported asset still carries that truth through `SceneImportService`
