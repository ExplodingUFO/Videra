# Requirements: Videra v1.23 Inspection Productization

**Defined:** 2026-04-21
**Core Value:** 跨平台 3D 渲染引擎的可靠性

## v1 Requirements

Requirements committed for the `v1.23` milestone. Each requirement maps to one roadmap phase.

### Inspection Session Contract

- [x] **INSP-01**: Host can capture and restore one typed inspection session through public APIs, including camera, object selection, host-owned annotations, measurement snap mode, clipping planes, and measurements.
- [x] **INSP-02**: Public docs and `Videra.InteractionSample` teach one canonical inspection-session flow without reaching into internal runtime or overlay seams.

### Inspection Replay and Support Truth

- [x] **INSP-03**: Replayable inspection bundle export/import preserves the same typed inspection session truth and remaps object-backed references correctly when it reloads a scene.
- [x] **INSP-04**: Inspection export/replay docs and tests describe one explicit artifact contract and replay limitations without turning `VideraView` into a broader project-format API.

## Future Requirements

Deferred after `v1.23`. These remain visible but are not part of the current roadmap.

### Performance and Packaging

- **PERF-01**: CI fails on agreed performance regressions and package-size budgets once the static asset/material path stabilizes.

### UI Validation

- **UIAD-01**: A second UI adapter proves the host seams outside Avalonia after the viewer path is stable.

### Advanced Runtime Features

- **ADV-01**: Lighting expansion, shadows, environment maps, transparent-sorting overhauls, post-processing, and animation remain an opt-in `2.0` line rather than a `1.x` runtime-core requirement.

## Out of Scope

Explicitly excluded from `v1.23` to keep the milestone focused on a coherent inspection product flow.

| Feature | Reason |
|---------|--------|
| Editor-style project files or broader workspace persistence | `v1.23` only productizes the shipped inspection session and replay/export flow |
| New backend work such as `OpenGL` / `WebGL` | Backend expansion still ranks below deepening the native viewer/runtime product |
| A second UI adapter | Avalonia remains the only validated UI shell in this milestone |
| Lighting-system expansion, animation, or broader advanced-runtime features | Advanced runtime features remain a later milestone instead of inflating the inspection product line |
| Generic `Chart3D` extraction or new chart product lines | `v1.23` is about the viewer inspection product, not widening the chart surface |
