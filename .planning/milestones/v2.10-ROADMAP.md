# Roadmap

## Active Milestone

`v2.10 Imported Asset Runtime Tightening`

Goal: tighten the imported-asset-to-runtime hot path so upload/render truth becomes more primitive-centric and scene-delta/upload behavior becomes more granular without widening runtime/product scope.

## Scope Guard

- No animation, lighting, shadows, environment maps, or post-processing breadth.
- No new chart family, chart kernel widening, extra UI adapter, or backend/API breadth.
- No compatibility shims, migration adapters, fallback runtime paths, or transitional contracts.
- No public package publishing or repository release tag creation.
- `.planning` remains local-only and does not define a repository release boundary by itself.
