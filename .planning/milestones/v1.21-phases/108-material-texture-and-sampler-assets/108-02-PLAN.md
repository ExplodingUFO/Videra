# Plan 108-02

## Goal

Define `Texture2D` and `Sampler` runtime assets with explicit ownership and reuse semantics independent from importer-specific types.

## Tasks

1. Add texture and sampler identity/contract types under `Videra.Core.Scene`.
2. Keep the contract direct: width, height, encoded image content format, image bytes, and sRGB intent for textures; explicit filter/wrap modes for samplers.
3. Extend tests so scene assets retain texture and sampler catalogs as first-class data.
