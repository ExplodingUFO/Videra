## Cross-Phase Integration

### Summary

1. `SceneDocument` now owns viewer scene truth directly, and runtime mutations no longer depend on rebuilding that truth from `Engine.SceneObjects`.
2. Import and upload are now separate steps: single and batch model load paths retain backend-neutral imported assets, and batch replace is atomic when every import succeeds.
3. Runtime rebind paths now rehydrate scene resources from retained assets through the active resource factory, instead of treating `SoftwareResourceFactory` as the steady-state truth.
4. Built-in backends now satisfy `IGraphicsDevice` / `IRenderSurface` directly, while demo/docs/repository guards all describe the same narrow scene-pipeline lab and fallback story.
