<objective>
在 Phase 16 的 renderer seam 之上接入真实 GPU backend，并让 `SurfaceChartView` 获得 chart-local native host shell、显式 fallback truth 与只读 rendering status。

这一步必须把 “GPU-first” 做成真实行为，而不是 GPU 结果回拷到 bitmap 再冒充主路径。
</objective>
