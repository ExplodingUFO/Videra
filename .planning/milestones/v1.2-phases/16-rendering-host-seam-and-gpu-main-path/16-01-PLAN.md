<objective>
建立 Phase 16 的第一层基础设施：新增 chart-specific rendering package，把 renderer ownership 从 `SurfaceChartView` 抽到 `SurfaceChartRenderHost`，并让现有 software path 先以 backend adapter 的形式继续可用。

目的不是马上做完 GPU，而是先把正确的 ownership seam 和 typed status/fallback vocabulary 落下来。
</objective>
