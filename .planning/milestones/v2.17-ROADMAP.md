# Roadmap

## Active Milestone

`v2.17 Bounded Emissive/Normal-Map Renderer Consumption Baseline`

Goal: consume the already-retained emissive and normal-map-ready material truth on the existing static-scene viewer/runtime path without widening into shadows, environment maps, post-processing, animation, new public package lines, or broader runtime abstraction work.

## Scope Guard

- No shadows, shadow maps, environment maps, post-processing, fullscreen pass framework, or new render-target/resource-set abstraction.
- No generic lighting framework, material-system rewrite, or broader shader-system expansion.
- No animation, skeletons, morph targets, mixers, extra UI adapters, backend/platform expansion, importer breadth widening, or chart-family expansion.
- No compatibility shims, migration adapters, downgrade paths, or transitional dual-path renderer contracts.
- Proof remains repository-owned and limited to the existing desktop hosts used for validation; every desktop program used in milestone validation must survive `10` seconds without crashing.
- `.planning` remains local-only and does not define a repository release boundary by itself.
