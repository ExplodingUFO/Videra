## Cross-Phase Integration

### Summary

1. Residency dirtiness now only transitions on scene mutation, resource-epoch change, or explicit rehydrate events; steady-state render cadence does not generate artificial upload work.
2. Imported assets and deferred `Object3D` instances now share a common `MeshPayload`, removing repeated vertex/index duplication while preserving reupload, wireframe, and picking behavior.
3. Upload drain now respects per-frame object and byte budgets and adapts between interactive and idle runtime modes using queue-aware heuristics.
4. Recovery and regression tests prove that retained scene truth rehydrates after backend changes without reimporting files and without bulk-uploading past the active frame budget.
