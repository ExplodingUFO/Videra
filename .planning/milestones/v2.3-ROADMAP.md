# Roadmap

## Active Milestone

### `v2.3 Surface Analytics Depth`

- Status: complete locally; awaiting next milestone definition
- Scope: deepen the existing surface family through explicit coordinate/scalar semantics, analysis-grade probe workflow, and one concrete analytics proof path without widening into another chart family or backend/platform breadth

## Operational Baseline

- `v2.2` is now archived locally as the shipped scatter-productization baseline.
- `SurfaceChartView` stays the active surface analytics boundary for this milestone.
- Generic `Chart3D`, `SurfaceMesh`, a fourth family, a new UI adapter, and `Wayland` / `OpenGL` work remain out of scope.
- `v2.3` is now closed locally; the next step is defining the next milestone rather than widening chart/runtime/platform scope implicitly.

- Goal: lock analytics-depth surface work before new surface contracts or analysis workflows start.
- Depends on: `v2.2` closeout

### Phase 145: Explicit Surface Coordinates and Scalar Separation

- Goal: add explicit/non-uniform coordinate semantics and keep height/color scalar fields separable under the existing surface family contracts.
- Depends on: Phase 144
- Status: `complete`

Plans:

- [x] add only the minimal data/axis/scalar hooks surface analytics actually needs
- [x] keep the existing surface path intact for regular-grid consumers
- [x] avoid widening into generic chart-family or backend abstractions

### Phase 146: Analysis-Grade Surface Probe Workflow

- Goal: add interpolated probe and comparative readouts on the existing Avalonia surface chart line.
- Depends on: Phase 145
- Status: `complete`

Plans:

- [x] add interpolation-grade probe results rather than tile-floor-only readouts
- [x] support one comparative probe workflow such as delta or paired readouts
- [x] keep the existing surface host shell and chart-local interaction boundary intact

### Phase 147: Surface Analytics Proof and Truth Guardrails

- Goal: prove one concrete analytics surface path and align docs/support/repository truth around it.
- Depends on: Phase 145 and Phase 146
- Status: `complete`

Plans:

- [x] add one concrete analytics proof path on the existing Avalonia surface line

- [x] update repository/CI guardrails without widening package or platform claims
