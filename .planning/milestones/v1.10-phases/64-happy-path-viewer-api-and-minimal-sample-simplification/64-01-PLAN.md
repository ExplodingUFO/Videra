# Phase 64 Plan 01

Create a narrow first-scene sample that demonstrates the default public `VideraView` flow without advanced extensibility seams.

## Steps

1. Add a dedicated `Videra.MinimalSample` project and wire it into the solution.
2. Keep the sample focused on `Options`, `LoadModelAsync(...)`, `FrameAll()`, `ResetCamera()`, and `BackendDiagnostics`.
3. Avoid `VideraView.Engine`, frame hooks, and pass contributors in the minimal sample code path.
