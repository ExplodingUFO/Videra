## Cross-Phase Integration

### Summary

1. Runtime scene truth is now authoritative at the document layer, with explicit entry identity, ownership, delta planning, and application seams instead of ad hoc engine-first synchronization.
2. GPU upload now flows through explicit residency state and a budgeted frame-prelude queue, so public scene mutation APIs no longer perform synchronous device work.
3. Backend or surface recovery now dirties and reuploads retained imported/runtime-owned entries against the new device without reimporting files or rebuilding scene truth.
4. The Scene Pipeline Lab, diagnostics shell, README, architecture docs, sample docs, and repository guards all describe the same narrow scene-pipeline contract.
