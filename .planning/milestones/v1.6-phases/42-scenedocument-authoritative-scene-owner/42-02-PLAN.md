# Plan 42-02

## Goal

Make viewer scene mutations document-first across object APIs and `Items` binding.

## Files

- `src/Videra.Avalonia/Runtime/VideraViewRuntime.Scene.cs`
- `src/Videra.Avalonia/Runtime/VideraViewRuntime.cs`

1. `AddObject`, `ReplaceScene`, `ClearScene`, and `Items` updates change the document first.
2. Item binding changes are honored before the render session reaches ready state.
