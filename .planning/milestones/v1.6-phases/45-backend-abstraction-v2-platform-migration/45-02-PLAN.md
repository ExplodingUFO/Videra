# Plan 45-02

## Goal

Let built-in D3D11, Vulkan, and Metal backends satisfy the device/surface contracts directly.

## Files

- `src/Videra.Platform.Windows/D3D11Backend.cs`
- `src/Videra.Platform.Linux/VulkanBackend.cs`
- `src/Videra.Platform.macOS/MetalBackend.cs`

## Acceptance

1. Built-in native backends implement `IGraphicsDevice` and `IRenderSurface` directly.
2. Each backend exposes device preference, surface state, and direct frame-context creation.
3. The migration does not widen viewer public API.
