# Requirements: Videra v1.24 Performance and Size Gates

**Defined:** 2026-04-21
**Core Value:** 跨平台 3D 渲染引擎的可靠性

## v1 Requirements

Requirements committed for the `v1.24` milestone. Each requirement maps to one roadmap phase.

### Performance Benchmark Contract

- [x] **PERF-01**: Repository-owned benchmark and measurement paths publish one stable evidence set for the shipped viewer workflows, including first-frame and steady interaction slices.
- [x] **PERF-02**: SurfaceCharts performance evidence includes a stable probe-or-interaction slice so chart regressions are judged on the same benchmark truth as the viewer path.

### Enforced Performance Gates

- [x] **PERF-03**: CI fails when agreed benchmark metrics regress beyond committed thresholds instead of relying on label-gated manual review.

### Shipping and Package Gates

- [x] **SHIP-01**: CI enforces explicit package-size budgets for the public package line and emits per-package size deltas in build artifacts or logs.
- [x] **SHIP-02**: Release-oriented smoke validation proves the shipped viewer stack can publish and start through the supported publish path without reopening package-boundary ambiguity.

## Future Requirements

Deferred after `v1.24`. These remain visible but are not part of the current roadmap.

### SurfaceCharts Productization

- **CHRT-01**: `Videra.SurfaceCharts` graduates from source-first evaluation into an explicit product line with stable packaging and chart-kernel boundaries.

### UI Validation

- **UIAD-01**: A second UI adapter proves the host seams outside Avalonia after the viewer path is stable.

### Advanced Runtime Features

- **ADV-01**: Lighting expansion, shadows, environment maps, transparent-sorting overhauls, post-processing, and animation remain an opt-in `2.0` line rather than a `1.x` runtime-core requirement.

## Out of Scope

Explicitly excluded from `v1.24` to keep the milestone focused on measurable shipping gates.

| Feature | Reason |
|---------|--------|
| New graphics backends such as `OpenGL` or `WebGL` | Backend expansion still ranks below enforcing the shipped native viewer/runtime line |
| A second UI adapter | `v1.24` is about objective shipping gates on the existing Avalonia product path |
| New chart products or generic `Chart3D` extraction | This milestone hardens release and performance truth instead of widening product scope |
| Lighting-system expansion, shadows, post-processing, or animation | Advanced runtime breadth remains a later product milestone, not a release-gate milestone |
