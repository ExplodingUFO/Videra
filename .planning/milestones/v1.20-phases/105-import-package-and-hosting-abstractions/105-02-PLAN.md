# Plan 105-02: Guard the Public API Boundary

## Objective

Prove that `Videra.Avalonia` stays a thin public shell and does not leak internal host/runtime/import seam types through its public API.

3. Keep the guard focused on actual product-boundary risks, not cosmetic type-shape churn.
