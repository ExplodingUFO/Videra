# 历史归档

本目录用于存放不再作为对外入口维护的历史文档。

这些文档仍然有参考价值，但其定位已从“当前说明”调整为“过程记录”或“阶段快照”。

## 归档分类

- `status/`: 阶段性状态、迁移完成说明、一次性总结
- `plans/`: 历史设计方案与规划文档
- `engineering/`: 工程性专项记录，例如测试稳定化过程
